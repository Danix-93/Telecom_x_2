# Predicción de Cancelación de Clientes

## Descripción
Proyecto de análisis y predicción de cancelación de clientes usando modelos de machine learning.

## Modelos
- DummyClassifier
- DecisionTreeClassifier
- RandomForestClassifier

## Resultados
| Modelo | Accuracy |
|------|------|
Dummy | 0.73 |
Árbol | 0.72 |
Random Forest | 0.78 |

## Tecnologías
Python, Pandas, Scikit-learn.
